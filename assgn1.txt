Git commands and its uses-

1.git init-  This command turns a directory into an empty Git repository. This is the first step in creating a repository.

2.touch-  to create file.

3.git status-  This command returns the current state of the repository.

4.git add-  to add files.

5.git config-  Record the changes made to the files to a local repository. For easy reference, each commit has a unique ID.

6. git config --local user.name= to addd name of user.

7 git config --local user.email=  To add email of user.

8. git push=   Sends data from local to onilne repositary.

9. git remote=  To connect a local repository with a remote repository.

10. git merge=  Integrate branches together. git merge combines the changes from one branch to another branch.
